# SupplementaryMaterialsForCowLocalizationPaper
The depository contains description of software developed and used for the paper “Monitoring of cow location in barn by an open source low cost low energy Bluetooth tag system”. The detailed instructions for getting results described in the paper are provided in the file CowBhave Localization Instructions.docx.
The software was developed on Matlab and duplicated on Julia.

The data is available at https://zenodo.org/.
